# NodeJS-api
API criada com NodeJS usando o framework express

Projeto criado para estudar a implementação de um projeto prático com Node JS

Para testar o Projeto, tenha instalado no computador o banco de dados MongoDB, inicialize o processo do MongoDB acessando o terminal do computador e digite o comando: mongod

> Baixe o projeto para alguma pasta do computador
> Instale o banco de dados MongoDB
> Inicialize o banco de dados a partir do terminal CMD com o comando: mongod
> Abra outra aba do terminal CMD e navegue até a pasta do projeto
> Inicialize a API com o comando: node app.js
> Instale o programa POSTMAN
> Faça requisições dos tipos (GET, POST, PUT, DELETE) para a url (http://localhost:1111/tarefa).
